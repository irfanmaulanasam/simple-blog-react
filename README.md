# simple-blog-react
making express server for react blog  in simplest way
its have 2 section 
  -client
  -server
 to starting the project use 
    npm i 
    (that is acronyme for npm install all dependencies in the project) 
    at each section
 to start client 
  use "npm start"
 to start server 
  use "nodemon app.js"
the database schema for article is 

  | key | types | description |
  |---|---|---|
  | title   | string |   |
  |  body | string |   |
  | author  | string |   |
